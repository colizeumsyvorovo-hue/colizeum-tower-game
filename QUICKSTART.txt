   - Личный кабинет

   - Новогодний дизайн
   - Интеграция с API

✅ Telegram Bot
   - Команды: /start, /stats, /help
   - Запуск игры через Web App

🚀 БЫСТРЫЙ ЗАПУСК:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Установите зависимости:
   cd server
   npm install

2. Создайте Telegram бота:
   - Откройте @BotFather в Telegram
   - Отправьте /newbot
   - Скопируйте полученный токен

3. Создайте файл .env в корне проекта:
   PORT=3000
   TELEGRAM_BOT_TOKEN=ваш_токен_здесь
   JWT_SECRET=colizeum-secret-key-change-in-production
   DATABASE_PATH=./database/game.db
   FRONTEND_URL=http://localhost:3000

4. Запустите сервер:
   cd server
   npm start

5. Откройте бота в Telegram и отправьте /start
